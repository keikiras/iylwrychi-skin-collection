# Reshiram Skin Collection
Iylwrychi / Reshiram / YGYoshi/ Krakence / P. Diddy / Samuel Epstein Private Skin Collection
sorted in mostly chronological order

# Why release these?
Sam was most known for the skins he used, and how gatekept they were. By releasing them to the wider public it allows many people to use these osu! skins for their own purpose, whether gameplay or inspiration for their own skin
